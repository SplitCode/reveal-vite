WHAT IS

CSS GRID

Grid Layout is a special CSS3 module that allows you to position elements in the form of a grid or table.

Basically, a grid could be broken down to two elements: the grid container and the grid-items.

BASIC

GRID LAYOUT

GRID

LINE

Grid line: the dividing line forming the grid structure. It can be either vertical (grid line of a column) or horizontal (grid line of a row).

GRID

TRACK

Grid track: the space between 2 adjacent grid lines. Essentially your columns and rows.

GRID

CELL

Grid cell: the space between 2 adjacent grid column lines and 2 adjacent grid row lines

GRID

AREA

Grid area: the space between any 4 grid lines. It can cover any number of rows and columns.

GRID

CONTAINER PROPS

To create a grid container, assign one of two values to its display style property: grid or inline-grid.

DISPLAY

GRID

The grid value defines the container as a block element 

.grid-container {
      display: grid;
  }

DISPLAY

INLINE-GRID

The inline-grid value defines the element as inline 

.grid-container {
      display: inline-grid;
  }

ROWS &

COLUMNS

Defines the columns and rows of the grid with a space separated list of value. 

.grid-container {
    grid-template-columns:  1fr 1fr 1fr 1fr;
    grid-template-rows:  1fr auto 2fr;
}

or

.grid-container {
  grid-template:  1fr auto 2fr  / 1fr 1fr 1fr 1fr;
}

GRID

Use the grid-template-columns style property to define columns.

            }

Use the grid-template-rows style property to define rows. 

            .grid-container {

                grid-template-rows: 4em 5em;

NAMED

LINES

The ability to explicitly name grid lines using square brackets

.grid-container {
  grid-template-rows: [start] 250px [line2] 400px [line3] 600px [end];
  grid-template-columns: 1fr 1fr [content-start navbar-end] repeat(10, 1fr);
}

FRACTIONAL

(FR) UNITS

fr (from fraction) is responsible for the free space inside the grid container.

.grid-container {
    grid-template:  1fr auto 2fr  / 1fr 1fr 1fr 1fr;
}

REPEAT()

FUNCTION

Allows to easily repeat settings.

.grid-container {
    grid-template:  1fr auto 2fr  / 1fr 1fr 1fr 1fr;
}

=

.grid-container {
	grid-template:  1fr auto 2fr / repeat(4, 1fr);
}

MINMAX

FUNCTION

Defines a size range greater than or equal to min and less than or equal to max. 

.grid-container {
    grid-template-columns:  repeat(3, minmax(100px, 1fr));
}

GRID

GAPS

Defines the space between grid tracks. 

.grid-container {
    row-gap: 5px;
    column-gap: 10px;
}

or

.grid-container {
    gap: 5px, 10px;
}

JUSTIFY ITEMS /

Justify-items aligns a grid-items on the row axis, align-items aligns items on the column axis, inside grid cells. 

justify-items: start;
justify-items: center;
align-items: end;
align-items: stretch;

JUSTIFY CONTENT /

ALIGN CONTENT

Justify-content align the grid on the row axis, and align-content align the grid on the column axis.

justify-content: center;
align-content: space-around;
align-content:space-evenly;

KEY

POINTS

  • Grid Layout in CSS is a flexible markup in which elements are arranged in a table. The table consists of cells that can be assembled into whole areas.
  • To create a grid, we need to add the line display: grid or inline-grid in the CSS markup file.
  • The grid consists of a parent container and its nested elements. Elements can occupy multiple cells that are stacked into columns, columns, or areas.

KEY

POINTS

  • Fraction is a unit of measurement in CSS, created for convenience â€” so that you can set the ratios between the elements and not manually adjust the percentages.
  • The main properties of the grid container are: grid-template, grid-template-columns, grid-template-rows, grid-template-areas. They allow to set the sizes of columns, rows, or entire areas.
  • The column-gap property is responsible for the indentation between the columns of the grid, and rowâ€”gap is responsible for the indentation between the rows.

KEY

POINTS

  • We can align container elements horizontally using justify-items and justify-content, and vertically with align-items and align-content.
  • We can adjust the location of elements from a specific grid line using the properties: grid-column-start, grid-column-end, grid-column, grid-row-start, grid-row-end, grid-row or a common grid-area property.
  • We can align a specific grid element using justify-self (horizontally) and align-self (vertically).
